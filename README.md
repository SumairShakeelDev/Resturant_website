# 🍽️ Restaurant Website

A modern, responsive Restaurant Website built using HTML, CSS, JavaScript, and Bootstrap. This project showcases a stylish UI for displaying menu items, special dishes, and customer interaction features.

## 🚀 Live Demo
https://resturant-website-hazel.vercel.app/

## 📌 Features
- 🏠 Beautiful Home Page with modern UI  
- 🍔 Interactive Menu Section (Veg / Non-Veg filter optional)  
- ⭐ Special Dishes Highlight Section  
- 📱 Fully Responsive Design (Mobile + Desktop)  
- 🎨 Clean and attractive Bootstrap layout  
- 📩 Contact Form for customer inquiries  
- ⚡ Smooth animations using JavaScript  

## 🛠️ Built With
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Bootstrap 5
